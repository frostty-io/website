## Quick Start

1. Download the latest macOS build from [GitHub Releases](https://github.com/doggo-sh/doggo/releases/latest).
2. Open the `.dmg` and drag Doggo to `Applications`.
3. Launch Doggo and create your first profile in Settings.
4. Open a tab, split it with `Cmd + D`, and start working.
