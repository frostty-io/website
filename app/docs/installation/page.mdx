# Installation

## macOS Download

1. Go to [Frostty Releases](https://github.com/frostty-io/frostty/releases/latest).
2. Download the latest macOS artifact (`.dmg` or `.zip`).
3. If you download `.zip`, extract it and move `Frostty.app` to `Applications`.
4. If you download `.dmg`, open it and drag Frostty to `Applications`.

## First Launch on macOS

1. Open Frostty from `Applications`.
2. If macOS blocks launch, open **System Settings â†’ Privacy & Security** and click **Open Anyway** for Frostty.
3. Re-open the app and confirm the prompt.

## Initial Setup

After launch:

1. Open **Settings** (`Cmd + ,`).
2. Create your first profile (name, shell, and starting directory).
3. Open a new tab (`Cmd + T`) and verify your shell + working directory are correct.

## Update to New Versions

1. Download the newest macOS build from Releases.
2. Replace the existing app in `Applications`.
3. Re-open Frostty; your settings and profiles remain available.
