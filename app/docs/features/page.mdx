# Features

Doggo ships with a broad set of functionality:

- Vertical tab bar with drag-and-drop tab ordering
- Split panes with resize handles and keyboard navigation
- Command palette and project palette
- Git status and branch operations from the UI
- AI mode for natural-language command generation
- Profiles for shell, cwd, and tab color preferences
- Session restoration across app restarts
- VS Code and Cursor integration
- GPU-accelerated terminal rendering
